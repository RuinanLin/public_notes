* 第一个判断所输出的话真的正确吗？由于你的＂第一个参数＂不是用户输入的第一个参数，把它改正．

	第一个（即0号）参数是可执行文件名，不是用户输入的第一个参数．进行如下改动：

	```
	7c7
	< 		printf("You only have one argument. You suck.\n");
	---
	> 		printf("You have no argument. You suck.\n");
	11c11
	< 		for (i = 0; i < argc; i++) {
	---
	> 		for (i = 1; i < argc; i++) {
	```

	首先是修改0个参数时的提示语，其次是当符合要求的时候不再给出文件名．
