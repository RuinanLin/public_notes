* 上网浏览不同的人喜欢什么样的函数格式．永远不要使用＂K&R＂语法，因为它过时了，而且容易使人混乱，但是当你碰到一些人使用这种格式时，要理解代码做了什么．

	ANSI/ISO-C风格看起来像这样：

	```c
	int add(int x, int y) {
		return x + y;
	}
	```

	而问题中提到的K&R风格看起来像这样：

	```c
	int add(x, y)
		int x;
		int y;
	{
		return x + y;
	}
	```
